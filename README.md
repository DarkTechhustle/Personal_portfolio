# Portfolio Website

A modern, responsive 3-page portfolio website to showcase your projects, skills, and experience.

## 📁 Project Structure

```
portfolio-website/
├── index.html       # Home page - Hero section & featured projects
├── about.html       # About page - Bio, skills, and experience
├── contact.html     # Contact page - Contact form & information
├── styles.css       # Main stylesheet with responsive design
├── script.js        # JavaScript for interactivity
└── README.md        # This file
```

## 🎨 Features

- **Responsive Design**: Mobile-first approach with breakpoints for tablets and desktops
- **Modern UI**: Clean, professional design with smooth animations
- **Navigation**: Sticky navbar with active page highlighting
- **Interactive Elements**: Hover effects, form validation, and scroll animations
- **SEO Ready**: Semantic HTML structure with proper meta tags
- **Smooth Scrolling**: Enhanced user experience with smooth navigation
- **Contact Form**: Functional form with validation

## 📄 Pages

### Home (index.html)

- Hero section with call-to-action
- Featured projects showcase
- Contact CTA section

### About (about.html)

- Personal bio
- Skills organized by category (Frontend, Backend, Tools)
- Experience timeline

### Contact (contact.html)

- Contact form with validation
- Contact information section
- Social media links

## 🚀 Getting Started

### Quick Start

1. Open `index.html` in your web browser
2. Navigate between pages using the navbar
3. Customize content with your own information

### Customization

**Update Your Information:**

- Edit the name and title in hero section
- Update project descriptions
- Modify skills and experience
- Add your contact details
- Replace social media links

**Styling:**

- Colors are defined in CSS variables (`:root`)
- Modify `--primary-color`, `--secondary-color`, etc.
- Adjust spacing, fonts, and sizes as needed

**Images:**

- Add project images by replacing `.project-image` divs
- Add a profile picture in the about section

## 🎯 Tips for Success

1. **Personalization**: Replace placeholder content with your real projects and experience
2. **Projects**: Add real links to your GitHub repositories or live demos
3. **Contact**: Replace email and phone with your actual contact information
4. **Social Links**: Add your actual social media profiles
5. **Mobile Testing**: Check all pages on mobile devices
